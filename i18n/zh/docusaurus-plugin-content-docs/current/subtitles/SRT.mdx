# SRT字幕格式

SubRip Text (SRT) 是一种字幕文本格式，被称为"所有字幕格式中最基础的格式"。SRT文件是扩展名为`.srt`的纯文本文件。

## 格式规范
字幕内容被组织成按顺序排列的组块（称为cue），每个组块包含开始和结束时间戳，格式为`小时:分钟:秒,毫秒`。注意毫秒值使用逗号分隔。开始和结束时间值之间用` --> `分隔。

### 非官方特性
部分播放器支持一些基础HTML标签，例如：
* `<b>粗体</b>` **粗体**
* `<i>斜体</i>` *斜体*
* `<u></u>` 下划线
* `<font color="#0000FF">蓝色</font>` 彩色文字

注意在不支持这些特性的播放器上，这些标签会直接显示为原始文本。

## 示例
```
1
00:00:00,000 --> 00:01:00,000
这条字幕将在视频流的第一分钟内显示

2
00:01:00,000 --> 00:01:30,000
这一条会在之后显示30秒
```
