# 7-zip (7z)

:::danger 需要协助

7-zip（7z）是一种支持多种数据压缩、加密及预处理算法的文件格式。该格式由自由开源的7-Zip压缩工具引入，该工具可处理包括[XZ](../data/xz.mdx)在内的多种压缩格式。

相比流行的[ZIP](../data/zip.mdx)格式，7-zip具有以下显著优势：
- 即使采用高效的ZIP编码器，7-zip工具仍能将文件压缩得"比ZIP格式小30-70%"。其核心采用LZMA和LZMA2算法，这些算法比DEFLATE更新且通常具有更好的压缩率
- 支持通过用户密码进行AES-256加密，安全性远高于ZIP常用的ZipCrypto加密方式
- 支持最大16EB的文件（传统ZIP限制为4GB，不过目前普遍采用的ZIP64已突破此限制）。7-zip还支持多种预处理滤镜，可提升可执行文件等特定类型数据的压缩效率

但7-zip也存在以下局限：
- 兼容性不及ZIP广泛，部分用户需要安装额外程序或插件才能解压7z文件
- 压缩/解压速度较慢。虽然7-zip工具在解码时实现了有效并行化，但这仅缩短实际处理时间而非用户等待时间，解压成本仍高于ZIP
- 缺乏内置的归档修复机制，用户需依赖第三方工具或备份副本恢复损坏的压缩包

macOS和多数Linux发行版已原生支持7z压缩包。
