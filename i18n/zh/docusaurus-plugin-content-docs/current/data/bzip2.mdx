# bzip2

:::danger 需要帮助
本节内容亟待完善。如果您能提供帮助，请查阅我们的[贡献指南](../contribution-guide.mdx)开始参与贡献！

bzip2 是一种开源的压缩文件格式及工具。其压缩效率略优于 [zip](./zip.mdx) 格式，但逊于基于 lzma 的压缩格式如 [xz](./xz.mdx) 和 [7z](./7z.mdx)。bzip2 无法同时压缩多个文件，如需压缩多个文件，应先将文件归档为 [tarball](./tar.mdx) 再使用 bzip2 进行压缩。
