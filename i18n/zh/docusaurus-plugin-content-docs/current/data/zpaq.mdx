# ZPAQ

ZPAQ 是一种结合多种技术实现高压缩率的无损数据压缩算法，由 Matt Mahoney 开发。

ZPAQ 采用多种不同的压缩算法，旨在实现最佳压缩比与压缩时间的平衡，同时尽可能生成最小体积的归档文件，而不过多考虑解压性能。根据 ZPAQ 官网描述，该算法专为"包含大量重复文件和已压缩文件的现实备份场景"设计。

ZPAQ 还被视作一种"增量日志归档器"，允许用户根据文件变更情况向现有归档中添加文件。对于备份场景而言，这能显著减少等待新备份完成的时间。由于 ZPAQ 极度注重压缩率，这一特性可有效缓解实际应用中因长时间压缩带来的负担。Windows 和 macOS 系统默认不支持 ZPAQ 归档格式，多数 Linux 发行版可能同样缺乏原生支持。
