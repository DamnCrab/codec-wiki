# XZ压缩格式

:::danger 寻求帮助
本节内容需要技术贡献。如果您能提供帮助，请查阅我们的[贡献指南](../contribution-guide.mdx)开始参与！
:::

XZ是一种基于Lempel-Ziv-Markov链算法(LZMA)的数据压缩格式及工具。XZ格式本身是对LZMA的改进，支持类似[7-zip](../data/7z.mdx)的预处理过滤器，可进一步提高压缩率。

XZ每次只能压缩单个文件，如需压缩多个文件，需要先使用[tar](./tar.mdx)打包。

相比其他压缩格式，XZ具有更广泛的系统支持，iOS、macOS和多数Linux发行版都默认包含解压工具。在Windows系统上，您可能需要安装7-Zip工具来处理XZ文件。

## 使用说明
以下说明基于Linux平台的`xz`工具，但同样适用于其他支持xz的平台。注意`xz`默认会在完成操作后删除原始文件，可通过下文参数禁用此行为。xz支持同时处理多个文件，会依次对每个文件执行指定操作。

### 压缩文件
```bash
xz {文件}
```
将在当前工作目录生成`{文件}.xz`压缩包。

高级参数示例：
```bash
xz -# --extreme -M 800Mib -T 2 -k {文件}
```

- `-#` 指定压缩级别(0-9)，0最快但压缩率最低，9最慢但压缩率最高
- `--extreme` 启用极限模式，突破标准预设的时间限制
- `-M {大小}` 限制内存使用量(可用百分比或绝对值表示)
- `-T {线程数}` 限制使用的CPU线程数
- `-k` 保留原始文件

### 解压文件
```bash
xz -d {文件}.xz
```
将xz压缩包解压为原始文件。

- `-M {大小}` 限制内存使用量
- `-T {线程数}` 限制使用的CPU线程数
- `-k` 保留压缩包文件
