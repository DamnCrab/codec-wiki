# 有损压缩

本条目内容不完整，正在完善中。

有损多媒体压缩通过永久性移除部分信息来减小多媒体文件的大小。该过程利用人类感官局限、保真度指标或吸引力指标，舍弃那些被认为次要的信息。有损压缩的目标是在保持预期质量水平的前提下减小多媒体文件体积。

## 有损压缩原理

有损压缩通过分析输入信号并移除次要部分来实现压缩，具体方法包括：

- **感知编码**（音频）：该技术会移除超出人类听觉范围或被其他声音掩盖的音频频率
- **量化**（音频）：此过程会降低某些音频成分的精度，可在不明显影响感知质量的情况下显著减小文件体积
