# 高动态范围成像

HDR（高动态范围）是现代电视和显示器中用于呈现更生动逼真画面的技术。简而言之，它能让您的电视显示比标准显示器更广的色域和亮度范围。这意味着您可以在图像的明亮和黑暗区域看到更多细节，使得电影、电视节目和电子游戏的画面显得更加真实。

HDR10通过元数据来指导电视以最佳方式呈现内容。这些元数据包含有关内容最高亮度级别和色域的信息，使电视能够根据显示内容调整设置。换言之，HDR10帮助电视呈现出更接近内容创作者初衷的画面，带来更具沉浸感的观看体验。

## 混合对数伽马（HLG）

HLG（混合对数伽马）是由BBC和NHK联合开发的一种HDR视频格式，专为同时优化标准动态范围（SDR）和HDR显示设备而设计。

要理解HLG的工作原理，需要了解人眼感知视频亮度与色彩的方式不同于其被捕获和在屏幕上显示的方式。亮度与色彩信息通常以对数曲线捕获，而SDR显示器通常使用伽马曲线再现图像。相比之下，HDR显示器使用另一种称为感知量化器（PQ）的曲线来再现图像。

HLG曲线是这两种曲线的混合体，这意味着它能同时优化SDR和HDR显示效果。该技术设计用于处理比SDR显示器更广的亮度范围，同时保持与SDR显示设备的向后兼容性。

简而言之，HLG曲线是一种视频采集与显示技术，能同时在SDR和HDR显示器上呈现优异效果。它如同连接视频采集与显示的桥梁，专为比传统SDR视频更广的亮度范围优化。最终结果是视频内容在SDR和HDR显示器上都能呈现更真实生动的画面。

## HDR10标准

HDR10是由美国消费技术协会于2015年8月27日公布的开放式高动态范围视频标准，是目前应用最广泛的HDR格式。该标准仅支持静态元数据。

## HDR10+技术

HDR10+本质上是HDR10的升级版，通过添加动态元数据支持（存储在``.json``文件中）来优化每个场景的内容亮度级别，实现导演的创作意图。

## 杜比视界

杜比视界是杜比实验室开发的专有HDR格式，直接与HDR10+形成竞争关系。
