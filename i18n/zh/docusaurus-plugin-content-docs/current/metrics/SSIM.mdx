# 结构相似性指数

结构相似性指数（SSIM）是一种全参考图像与视频质量评估指标，用于量化由有损压缩等处理导致的图像保真度下降。该指标于2004年发表在[*IEEE图像处理汇刊*](https://en.wikipedia.org/wiki/IEEE_Transactions_on_Image_Processing)上，通过评估人类自然感知视觉质量所依赖的结构信息，试图解决传统指标如[峰值信噪比（PSNR）](./PSNR.mdx)的局限性。

## 概述

SSIM通过比较原始图像与处理后图像之间的三个关键要素来工作：亮度、对比度和结构。亮度比较测量两幅图像平均像素强度的相似性。这三个比较结果被组合产生一个范围在-1到1之间的相似度分数，其中1表示完全的结构相似性。

SSIM的主要优势之一是其较传统指标能更好地符合人类视觉感知。SSIM认识到像素间存在强相互依赖性（尤其在空间距离接近时），因此特别擅长检测人类观察者会注意到的结构信息变化，如模糊、块状伪影或噪点。

作为视频编码器中改进决策的环内质量指标，SSIM比PSNR计算成本更高，且并不总能显著提升单位比特率下的保真度。在许多情况下仍需配合[心理视觉](../introduction/psychovisual.mdx)编码器选项才能实现最佳感知效率。

## 局限性

在多媒体压缩领域，当优化目标是保持特定文件尺寸下的最优感知质量时，SSIM更具价值。但SSIM与人类视觉系统并非完美契合，因此开发了[XPSNR](./XPSNR.mdx)和[SSIMULACRA2](./SSIMULACRA2.mdx)等新指标以获得更接近人类感知的相关性。为满足特定需求，现代已发展出多种SSIM变体和扩展：多尺度SSIM（MS-SSIM）通过不同尺度评估图像以更好匹配人类视觉；色彩SSIM变体能更有效处理颜色信息；SSIMULACRA（后被SSIMULACRA2取代）则用于提升与人类感知的相关性。这些改进使SSIM在不断变化的多媒体压缩领域中持续优化其感知目标。
