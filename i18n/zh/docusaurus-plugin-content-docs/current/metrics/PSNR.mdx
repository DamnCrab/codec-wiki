# 峰值信噪比(PSNR)

PSNR（峰值信噪比）是最广泛使用的客观全参考图像视频质量评估指标之一。它测量最大可能信号功率与破坏性噪声功率之间的比率，以分贝(dB)表示。该指标通过计算参考图像与失真图像之间的均方误差(MSE)得出。[维基百科PSNR条目](https://en.wikipedia.org/wiki/Peak_signal-to-noise_ratio)详细阐述了其理论基础和数学原理。

## 视频压缩领域应用

PSNR在视频编码应用中广泛使用，因其计算速度快，适合实时编码决策，并为不同编码方案比较提供了统一的数学基础。视频编码器内部也使用PSNR来辅助压缩决策。

### 编码器内部机制

在视频编码器中，PSNR在率失真优化(RDO)过程中起关键作用——这是寻找码率与质量最佳平衡的过程。编码器将PSNR作为环路指标来评估不同编码决策，如模式选择、运动估计和量化参数(QP)选择。

例如，当在不同预测模式或块大小之间做选择时，编码器会计算每个选项的PSNR影响及其比特成本来确定最优方案。

### 局限性

尽管PSNR因简单高效而被广泛采用，但它存在明显缺陷。该指标与人眼质量感知的相关性较差——PSNR对所有像素级误差高度敏感，而其中许多误差并无实际感知影响。当PSNR用于指导视频编码的RDO时，可能导致次优的质量决策。为此，编码器需要创造性地考虑人类视觉系统特性，我们在[心理视觉条目](../introduction/psychovisual.mdx)中有所阐述。

PSNR作为全参考失真指标的不足，促使了[SSIM](./SSIM.mdx)、[VMAF](./VMAF.mdx)和[XPSNR](./XPSNR.mdx)等更先进指标的发展，这些指标试图更好地模拟人类视觉感知。

## 实际应用场景

PSNR常用于评估图像视频压缩算法、流媒体质量分析、编解码器性能比较等场景。许多视频编码工具和分析套件会分别报告亮度/色度分量(Y、U、V)的PSNR值及加权平均值。尽管存在局限，PSNR仍是视频压缩领域的重要工具，特别是与其他质量指标和主观评估方法结合使用时。
