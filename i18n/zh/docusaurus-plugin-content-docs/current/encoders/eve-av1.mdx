# Eve-AV1

Eve-AV1是由Two Orioles有限责任公司开发的专有[AV1](../video/AV1.mdx)视频编码器，该公司以开发最广泛使用的AV1软件解码器[dav1d](../utilities/dav1d.mdx)而闻名。

根据Two Orioles公司[Eve-AV1产品页面](https://www.twoorioles.com/eve-av1)所述：

> Eve-AV1为流媒体点播视频提供了前所未有的画质水平。在相同视觉质量下，相比其他AV1编码器可降低20%的比特率。用户也可选择将编码速度提升3至5倍

该页面提供了一个视觉对比示例以及基于[VMAF](../metrics/VMAF.mdx)的BD-rate图表。

## 性能表现

![Eve-AV1 VMAF性能图表](/img/eve_av1_speed.webp)

从VMAF指标来看，Eve-AV1表现相当出色，且能达到令人印象深刻的编码速度。但由于缺乏更详细的视觉对比数据，这些性能指标仍需谨慎看待。
