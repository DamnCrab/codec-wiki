# x266

x266是即将推出的[H.266/VVC](/docs/video/VVC)编解码器软件编码器。目前关于该编码器的公开信息极少，仅知它仍由[MulticoreWare](https://multicorewareinc.com)开发中，且在其[2023年1月31日网络研讨会FAQ页面](https://multicorewareinc.com/faq-webinar/x266-faq.html)设有专门问答区。

根据其FAQ说明，2023年下半年曾是其v1.0公开版本的"非常粗略的预估发布时间"，但截至2024年10月14日尚未有任何更新发布。
