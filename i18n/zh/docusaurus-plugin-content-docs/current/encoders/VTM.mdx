# VTM

VTM是原始的[H.266](../video/VVC.mdx)（更广为人知的名称是VVC）参考编码器，与[VVenC](../encoders/VVenC.mdx)等替代方案竞争。如今，它与其他MPEG参考编码器（如[HM](../encoders/HM.mdx)和[JM](../encoders/JM.mdx)）一样，因使用复杂性和速度劣势而成为极少使用的特殊工具；不过由于当前无论选择哪种编码实现方案，VVC编码都存在难度，VTM可能会更有实用价值。x264的编码效率高于JM。
