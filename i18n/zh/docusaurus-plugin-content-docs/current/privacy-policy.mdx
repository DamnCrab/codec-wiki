# 隐私政策

本网站托管于GitHub Pages，网站使用受[GitHub隐私政策](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement)约束。GitHub可能会以日志文件形式存储您的访问信息。

此外，本网站使用位于新加坡的自托管[Plausible Analytics](https://plausible.io)实例。Plausible Analytics是一个轻量级开源网站流量分析平台。我们不会追踪、收集或存储任何个人身份信息；Plausible Analytics仅收集汇总数据，这些数据无法识别任何网站访问者身份。

收集这些信息是为了理解用户如何与Codec Wiki互动，从而改进我们的工作以更好地满足用户需求。

我们未使用Cookie技术。收集的信息包括：
- 页面URL
- HTTP来源地址
- 浏览器类型及版本
- 操作系统类型及版本
- 设备类型
- 国家、地区及城市

基于这些信息，我们承诺：
- 分析数据不会明确共享给广告公司或任何其他企业，所有数据均免费公开在我们的[分析页面](https://analytics.x266.mov/wiki.x266.mov/)
- 分析数据不会主动发送给任何第三方
- 不会将分析数据用于商业盈利

作为一个致力于普及多媒体编解码器知识的维基站点，分析数据通过我们的[分析页面](https://analytics.x266.mov/wiki.x266.mov/)向公众开放。

您可以通过多种广告/追踪拦截工具选择退出，例如[uBlock Origin](https://ublockorigin.com/)即可拦截我们的分析脚本。

如有任何疑问或建议，请联系网站维护者Gianni Rosato：[grosatowork@proton.me](mailto:grosatowork@proton.me)。
