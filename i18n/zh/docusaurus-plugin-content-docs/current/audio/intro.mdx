# 有损与无损音频压缩导论

:::info 内容维护中
本文档内容尚未完善，目前正在持续更新中。
:::

数字音频是将声音以数字化形式记录或转换的呈现方式。理解数字音频需要掌握几个核心概念，包括**采样**、**奈奎斯特频率**以及**奈奎斯特-香农采样定理**。

## 采样与奈奎斯特频率

**采样**是指通过以固定间隔测量音频信号的振幅，将连续的模拟音频信号转换为离散数字信号的过程。这个测量频率称为*采样率*，通常以每秒采样次数或赫兹(Hz)为单位。例如音频CD采用的44,100Hz采样率，表示每秒对音频信号进行44,100次采样。

要准确再现声波，每个周期至少需要两次测量：一次捕捉波峰，一次捕捉波谷。如果采样次数低于每个周期两次，就无法区分不同频率——这正是**奈奎斯特频率**概念的由来。以哈利·奈奎斯特命名的**奈奎斯特频率**是指*离散信号处理系统采样率的一半*。对于给定采样率，奈奎斯特频率代表了能够准确采样而不引入混叠失真的最高频率。例如48,000Hz采样率对应的奈奎斯特频率是24,000Hz。采集超过系统奈奎斯特频率的信号会导致**混叠**伪影，即高频成分在采样信号中表现为低频，造成信息失真。

**奈奎斯特-香农采样定理**指出：为避免混叠，采样率必须至少是信号最高频率的两倍。这一定理对于确保数字音频能完整保留原始模拟信号信息而不失真至关重要。

## 无损音频压缩

无损压缩的主要优势在于保持音频质量，因此特别适合专业音频制作、归档存储以及对高保真度有要求的场景。但无损文件体积明显大于有损格式，这对存储和（特别是）流媒体传输构成限制。

## 有损音频压缩

有损压缩的主要优势是大幅减小文件体积，因此特别适合流媒体、移动设备及存储空间有限的场景。但代价是可能损失音频质量，在高要求的听音环境中尤为明显。
