# MP3

MP3，全称为MPEG-1 Audio Layer III或MPEG-2 Audio Layer III，是一种数字音频编码格式。该技术主要由德国Fraunhofer研究所在Karlheinz Brandenburg的领导下开发，并得到其他国家数字科学家的支持。

MP3规范定义于两个ISO/IEC标准系列：MPEG-1: 11172-3和MPEG-2: 13818-32。它采用有损压缩技术，相比未压缩音频通常能显著减小文件体积。

MP3的有损压缩原理是通过降低（或近似处理）人耳听觉阈限之外的音频成分精度（基于[心理声学](../introduction/terminology.mdx#perceputal--psychovisual--psychoacoustic)分析），仅存储关键频段对应的系数。

相较于CD音质的数字音频，MP3压缩通常可实现75%至95%的体积缩减。例如，以128kbit/s恒定比特率编码的MP3文件，其体积约为原始CD音频的9%。当比特率达到320kb/s时，MP3音频可达到听觉无损级别。

尽管该格式标准于1993年定稿（1995年修订加入对更低采样率和比特率的支持），MP3文件至今仍随处可见。这是因为MP3堪称首个被广泛采用的高音质小型化音频格式。与[Opus](../audio/Opus.mdx)、[AAC](../audio/AAC.mdx)等现代格式相比，MP3或许不再出众，但其仍被众多设备和软件广泛支持，无疑为现代编解码器树立了难以超越的行业标杆。
