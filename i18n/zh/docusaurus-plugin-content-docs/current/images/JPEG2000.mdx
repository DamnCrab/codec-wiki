# JPEG 2000

JPEG-2000是一种基于小波技术的早期图像压缩格式，能在保持图像质量的同时实现高压缩比。该格式同时支持有损和无损压缩，广泛应用于数码摄影、医学影像和视频监控等领域。通过多种软件工具和库即可实现JPEG-2000文件的透明压缩与解压，使其成为灵活且广泛支持的图像存储与传输格式。JPEG-2000未能在互联网领域有效普及，但在数字影院发行领域经常使用该格式。"DCP"(数字影院包)是影院用于发行和播放数字电影的标准格式，这类DCP通常采用JPEG-2000进行无损压缩。
