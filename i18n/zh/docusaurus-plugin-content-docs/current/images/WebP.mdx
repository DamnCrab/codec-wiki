# WebP

WebP是由Google于2010年首次发布的免费图像文件格式。它包含两种主要工作"模式"：基于[VP8](../video/VP8.mdx)视频编解码器的有损模式，以及2011年新增的无损压缩模式。

## 编码方法

### 使用libwebp
libwebp支持WebP、JPEG、PNG、PNM（PGM、PPM、PAM）和TIFF作为输入格式，质量参数（`-q`）取值范围为0（质量最低，文件最小）到100（质量最高，文件最大）。如需使用无损模式，需改用`-z`参数，数值表示压缩强度，范围从0（编码最快，文件最大）到9（编码最慢，文件最小）。

```bash
cwebp example.png -q 75 -o example.webp
```

## 解码方法

```bash
dwebp example.webp -o example.png
```
