# GIF

本节内容需要技术贡献。如果您能提供帮助，请查阅我们的[贡献指南](../contribution-guide.mdx)开始参与贡献！

图形交换格式(GIF)是由CompuServe于1987年首次发布的图像文件格式。尽管其编码效率已过时，但由于对动画图像的广泛支持，至今仍广受欢迎。后续出现的动态图像格式如动画[AVIF](../images/AVIF.mdx)、动画[WebP](../images/WebP.mdx)以及动画[PNG](../images/PNG.mdx)变体APNG，在功能上均已超越GIF。
