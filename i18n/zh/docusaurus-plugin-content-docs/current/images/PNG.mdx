# PNG

便携式网络图形（PNG）是1996年发布的免费无损图像文件格式。它最初作为[GIF](./GIF.mdx)的替代方案被创建，当时GIF还属于专利格式。2008年通过APNG的发布获得了类似GIF的动画支持，目前所有主流网页浏览器均已支持APNG。
