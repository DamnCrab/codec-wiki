# HEIC 图像格式

:::danger 需要协助
本节内容亟待完善。如果您能提供帮助，请查阅我们的[贡献指南](../contribution-guide.mdx)开始参与贡献！
:::

HEIC（高效图像格式）是一种新型图像编解码器，相比传统格式如[JPEG](./JPEG.mdx)能提供更优的压缩效率和性能表现。HEIC文件内部采用[HEVC](../video/HEVC.mdx)编码技术，这意味着该格式需要支付专利授权费。虽然这一特性限制了其在Web领域的普及，但该格式已获得包括整个苹果生态系统在内的众多现代设备支持。iPhone默认使用其HEVC硬件视频编码器拍摄HDR格式的HEIC照片。部分安卓设备虽然也能拍摄HEIC格式，但通常由JPEG转码而来。HEIC现已被采用相同容器存储[AV1](../video/AV1.mdx)压缩图像的[AVIF](./AVIF.mdx)格式大幅超越。

*8位/通道，10位/通道*

是否支持动画？*是*

是否支持透明度？*是*

是否支持渐进式解码？*否*

是否免专利费？*否*
