# eac3to

eac3to是由madshi开发的命令行工具，主要用于音频转换和处理未加密的蓝光原盘(BDMV)。

:::info 兼容性说明
该软件仅支持Windows系统，但可通过Wine兼容层运行
:::

:::caution 尽量避免使用
除非某些古老[私用种子站](https://wiki.installgentoo.com/wiki/Private_trackers)的特殊要求，否则没有任何理由使用该软件——因为它需要大量未随软件包分发的依赖项。建议改用FFmpeg等替代方案。
:::

## 安装方法

- 从[VideoHelp](https://www.videohelp.com/software/eac3to)下载，拖放至目标文件夹，必要时添加至PATH环境变量

## 使用方法

### 音频转换

示例：
```bash title="将PCM音频转为FLAC格式"
eac3to 源文件.pcm 输出文件.flac
```

```bash title="转换TrueHD至FLAC并同时提取AC-3兼容层"
eac3to 源文件.thd 输出文件.flac 输出文件.ac3
```

### 获取BDMV信息

```bash
eac3to Movie.2024.Bluray/BDMV
```

### 分离音视频流

```bash
eac3to Movie.2024.Bluray/BDMV -demux
```

### 音频延时处理
```bash
eac3to 输入.eac3 输出.eac3 1000ms
```
