# dovi_tool

**dovi_tool** 是一个用Rust编写的命令行工具，集成了多种用于处理杜比视界(Dolby Vision)的实用功能。

## 安装指南

<Tabs>
    <TabItem value="unixlike" label="Linux & macOS">
        您可以从[GitHub Releases](https://github.com/quietvoid/dovi_tool/releases)下载适用于Linux和macOS的官方预编译二进制文件，或按照以下说明自行编译该工具。

        请确保已安装Rust环境。

        ```bash
        git clone https://github.com/quietvoid/dovi_tool.git
        cd dovi_tool
        RUSTFLAGS="-C target-cpu=native" cargo build --release
        ```
    </TabItem>
    <TabItem value="windows" label="Windows">
        您可以从[GitHub Releases](https://github.com/quietvoid/dovi_tool/releases)下载适用于Windows的官方预编译二进制文件，或按照以下说明自行编译该工具。

        请确保已安装Rust环境。

        ```pwsh
        git clone https://github.com/quietvoid/dovi_tool.git
        cd dovi_tool
        set RUSTFLAGS=-C target-cpu=native 
        cargo build --release
        ```
    </TabItem>
</Tabs>

## 使用说明

```bash title="通用命令格式"
dovi_tool [OPTIONS] <SUBCOMMAND>
```

```bash title="查看子命令的详细选项"
dovi_tool <SUBCOMMAND> --help
```
