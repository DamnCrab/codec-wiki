# 资源

您可能会用到的视频编码资源。

- https://guide.encode.moe - 滤镜处理与字幕压制
- https://lvsfunc.encode.moe/en/latest - lvsfunc 文档库
- https://silentaperture.gitlab.io/mdbook-guide/introduction.html - SilentAperture 的进阶编码指南（主要关于滤镜处理）
- https://x265.readthedocs.io/en/master - x265 技术文档（由MulticoreWare官方编写）
- http://www.chaneru.com/Roku/HLS/X264_Settings.htm - x264 参数设置
- https://kokomins.wordpress.com/2019/10/10/anime-encoding-guide-for-x265-and-why-to-never-use-flac - Kokomins 的动画压制指南（包含关于心理视觉优化和x265的优秀建议）
- https://wiki.xiph.org/Main_Page - Xiph 基金会维基
