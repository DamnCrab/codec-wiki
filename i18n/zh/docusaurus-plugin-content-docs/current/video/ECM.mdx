# ECM增强压缩模型

:::danger 需要协助
本节内容亟待完善。如果您能提供帮助，请查阅我们的[贡献指南](../contribution-guide.mdx)开始参与贡献！

ECM（Enhanced Compression Model，增强压缩模型）是由弗劳恩霍夫海因里希-赫兹研究所开发的视频编解码器。据称该技术旨在接替H.266（即[VVC](../video/VVC.mdx)）标准。

其[Git仓库](https://vcgit.hhi.fraunhofer.de/ecm/ECM)提供了更多信息：

```md
# ECM参考软件

本软件包是增强压缩模型（ECM）的参考实现，包含编码器和解码器功能。
参考软件可帮助视频编码标准用户建立和测试兼容性、互操作性，同时用于技术演示和教育用途。
本软件由ITU-T视频编码专家组（VCEG，ITU-T第16研究组第6问题组）与ISO/IEC运动图像专家组（MPEG，ISO/IEC联合技术委员会第1分委会第29工作组11）联合开发。
```

除上述概述外，仓库的README文件中提供了构建说明。目前关于ECM的公开信息有限，仅知其很可能将成为H.267标准。
