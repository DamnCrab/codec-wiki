# VC-1 视频编解码器

VC-1是由微软开发的视频编解码器，于2006年发布，主要旨在与[AVC](../video/AVC.mdx)标准竞争。

## 编码方案
VC-1视频流可封装在`.mp4`、`.mkv`和`.avi`容器中。

## 解码支持
VC-1可通过[FFmpeg](../utilities/ffmpeg.mdx)、[VLC](../video-players.mdx)、[MPV](../video-players.mdx)等软件解码器进行解码，所有支持蓝光播放的设备也均具备VC-1解码能力。
