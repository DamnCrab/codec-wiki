# Theora

Theora是一款发布于2004年的传统视频编解码器，其技术源自VP3编解码器，与[VP8](./VP8.mdx)、[VP9](./VP9.mdx)及[AV1](./AV1.mdx)属于同源技术体系。

## 编码说明
Theora视频流通常封装在ogg容器（扩展名为`.ogg`或`.ogv`）中，并搭配[vorbis](../audio/Vorbis.mdx)音频编码。
