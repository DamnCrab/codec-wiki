# H.266

:::danger 急需协助
本节内容需要技术贡献。如果您能提供帮助，请参阅我们的[贡献指南](../contribution-guide.mdx)开始参与！
:::

H.266（又称VVC/Versatile Video Coding）是由联合视频专家组（JVET）于2020年标准化的新一代编解码器。作为[H.265](../video/HEVC.mdx)的继任者，其宣称能提升40%的编码效率。实际测试表明，采用[VVenC](../encoders/VVenC.mdx)编码器时，其效率与[AV1](../video/AV1.mdx)相当，但由于其算法复杂度更高，解码难度也更大。该标准受专利授权限制。

2024年4月发布的[FFmpeg](../utilities/ffmpeg.mdx) 7.0版本已内置原生VVC解码器支持。
