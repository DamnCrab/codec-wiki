# VP9

:::danger 需要协助
本节内容需要补充完善。如果您能提供帮助，请参阅[贡献指南](../contribution-guide.mdx)开始参与贡献！
:::

VP9作为YouTube多年来的首选编解码器而闻名，这是一种免版税的视频压缩格式，与[HEVC](../video/HEVC.mdx)和[AVC](../video/AVC.mdx)形成竞争关系。在视觉质量方面，VP9略逊于HEVC，但其计算复杂度低得多，因此在解码时对系统资源消耗非常友好。如果您对VP9编码感兴趣，请参阅[vpxenc](../encoders/vpxenc.mdx)或[SVT-VP9](../encoders/SVT-VP9.mdx)章节。
