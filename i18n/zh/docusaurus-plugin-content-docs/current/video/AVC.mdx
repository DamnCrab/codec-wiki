# H.264

H.264，又称AVC（高级视频编码），是一个在多媒体编解码技术发展历程中具有里程碑意义的视频压缩标准。相比前代标准，H.264凭借革命性的高效压缩性能，成为近十年来实际通用的视频压缩规范。其发展历程以持续优化和广泛适配为特征，特别是在高度优化的[x264编码器](../encoders/x264.mdx)中表现尤为突出。尽管问世已超过十年，H.264至今仍保持重要地位。虽然新一代编解码器如[H.265](../video/HEVC.mdx)、[VP9](../video/VP9.mdx)和[AV1](../video/AV1.mdx)在压缩效率上有所突破，但目前尚未达到H.264的普及程度。编解码器的选择主要取决于应用场景的具体需求、兼容性支持程度，以及压缩效率与计算复杂度之间的平衡关系。
