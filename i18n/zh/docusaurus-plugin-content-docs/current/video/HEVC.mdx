# H.265

H.265，全称为高效视频编码（High-Efficiency Video Coding，HEVC），是一种旨在以更小文件体积提供更高画质的视频压缩标准。作为[H.264](../video/AVC.mdx)编解码器的继任者，其开发初衷是为了满足日益增长的高效高质量视频压缩需求，特别是针对4K内容的应用场景。需注意该编解码器涉及专利授权费用。
