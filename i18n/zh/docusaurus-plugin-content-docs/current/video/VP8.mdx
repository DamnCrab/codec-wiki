# VP8

:::danger 急需协助
本节内容需要您的贡献。如果您能提供帮助，请参阅我们的[贡献指南](../contribution-guide.mdx)开始参与！
:::

VP8是由On2 Technologies开发并于2008年发布的视频压缩格式。2010年，Google将其作为免版税编解码器公开发布。其编码效率与[AVC](../video/AVC.mdx)相当。VP8曾是免版税编解码器领域的重要参与者，被广泛应用于网页视频传输等场景。它是[VP9](../video/VP9.mdx)和[AV1](../video/AV1.mdx)的前身，后两者都进一步提升了视频压缩效率。VP8因其规范文档混乱且被普遍认为不完善而受到批评。
