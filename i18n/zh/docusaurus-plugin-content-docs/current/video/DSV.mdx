# DSV 数字子带视频编解码器

DSV 或 DSV1（Digital Subband Video 1）是一款紧凑的纯软件视频编解码器，专注于基于小波变换的简易压缩技术，其功能特性可媲美早期MPEG标准。该编解码器的参考实现专为C语言环境的灵活性和易用性而设计。截至2024年10月，DSV1已被[DSV2](./DSV2.mdx)取代。

## 特性

- **小波压缩技术**：采用多分辨率子带分析（小波变换）替代DCT变换，更多技术细节请参阅[DSV2](./DSV2.mdx)页面
- **运动补偿**：支持半像素精度运动补偿
- **色度抽样**：支持4:1:1、4:2:0、4:2:2及4:4:4格式
- **自适应量化**：动态调整量化参数以提升画质
- **帧类型**：支持帧内编码与帧间编码，采用可变长度封闭式图像组（GOP）（无双向预测/B帧，仅前向预测）
- **熵编码**：采用交错指数哥伦布编码以简化实现
- **压缩范围**：相比MPEG-1/2支持更广的压缩范围与更低比特率

参考编码器具备以下功能：

- **码率控制**：支持单次平均比特率（ABR）或恒定速率因子（CRF）模式
- **人眼视觉优化**：简易的帧内块模式决策机制
- **场景切换检测**：基于平均亮度变化的检测算法
- **分层运动估计**：提升运动估计效率
- **自适应量化稳定性追踪**：改善编码质量
- **C89兼容性**：完全符合C89标准，无外部库依赖，纯整数运算，单线程设计，具备高度可移植性

## 技术限制

- 不支持隔行扫描
- 仅支持8位色深
- 帧尺寸必须能被2整除

### 使用示例

目前仅有参考编码器可用：

- 编码命令：
  ```
  ./dsv1 e -y -v -inp_bus_cif.yuv -out_saved.dsv -gop12 -w352 -h288 -fps_num30 -qp85 -kbps1100
  ```
- 解码命令：
  ```
  ./dsv1 d -inp_video.dsv -out_decompressed.yuv -out420p
  ```
