# 色彩范围

色彩范围是描述像素有效值的概念。
通常，RGB会使用全范围(full range)，而YUV会使用有限范围(limited range)。

这意味着什么？

对于8位视频，全范围表示可以使用0-255之间的所有值来表示颜色值。而有限范围则表示只有16-235（亮度）或16-240（色度）之间的值才是有效的，超出该范围的值将被截断以适配该范围。这些范围会按比例扩展到高位深视频。

为什么存在有限范围？本质上这是历史原因造成的，但如今已成为我们必须遵循的惯例。尽管全范围可能提供稍好的色彩精度，但对于高位深内容意义不大——甚至HDR蓝光碟也使用有限色彩范围。因此建议遵循现有惯例。
